###template_project

>–.git 通过git管理项目会生成这个文件夹  
>–node_modules 组件目录  
>–dist 发布环境  
  >>–css 样式文件(style.css style.min.css)  
  >>–images 图片文件(压缩图片)  
  >>–js js文件(main.js main.min.js)  
  >>–index.html 静态文件(压缩html)  

>–src 生产环境  
  >>–sass sass文件 
  >>–images 图片文件  
  >>–js js文件  
  >>–index.html 静态文件

>–.jshintrc jshint配置文件
>–gulpfile.js gulp任务文件
